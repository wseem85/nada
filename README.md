# Nada Art Marketplace

A full-stack art marketplace designed for artists and collectors to showcase, discover, and purchase digital and physical artwork.

Built with the MERN stack and modern web technologies, Nada provides a complete e-commerce workflow including authentication, artwork management, image processing, payments, and order tracking.

## Live Demo

https://nadaart.onrender.com/

## Features

### User Features

- Secure user registration and authentication
- User profiles and account management
- Browse artwork by category
- Search and filter artwork
- Shopping cart functionality
- Checkout and order management
- Purchase digital and physical artwork
- Artwork favorites and collections

### Artist Features

- Upload and manage artwork
- Image processing and cloud storage
- Artwork categorization and metadata management
- Inventory and availability control

### Admin Features

- User management
- Artwork moderation
- Order monitoring
- Platform administration dashboard

## Architecture Highlights

- JWT-based authentication and authorization
- Role-based access control
- RESTful API architecture
- Secure payment workflow using Stripe
- Cloud-based media storage with Cloudinary
- Separation of client and server applications
- Responsive user experience across devices

## Project Goals

The project was built to explore and implement real-world marketplace functionality including:

- Full-stack application architecture
